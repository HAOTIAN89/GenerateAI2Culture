The Model

The model we used is the combination of ImageBind unified latent space and Stable-Diffusion-2-1-unclip. It should be the most powerful multimedia search and generation open-source model which can run on the normal computers nowadays. 

The DataBase

The databases created were not used to train the model but to prove if it can understand the cultures around the world once provided a multimedia database of them.

Features

The model used can receive either image, text or audio as an input for the two functions that it supports which are search for images or generation of images. The inputs can be isolated or combined for generation.

Objective

The Objective of the project is to prove how far can an AI model understand the cultural differences among 2 cultures presented, we will try to run search/generation processes to prove its understanding.
